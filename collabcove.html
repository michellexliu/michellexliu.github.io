  • CollabCove

  • Workflow management for teams.

  • I built CollabCove for my Advanced Data Structures class's term project. CollabCove is a web application that allows teams to collaborate on group projects and organize their projects/tasks. It was coded wth PHP, HTML, CSS, and JavaScript.


    My contributions:

    • Login/registration system
    • Dark/light mode toggle
    • Project creation system

  • 01. Login and Registration

  • In the login and registration system, I used HTML, JavaScript, and CSS to develop the front-end and PHP and SQL to manage the back-end and user database. The user verification system was fully functional and successful at creating a user, storing user information in a database, and verifying user credentials. This project was my first time working with PHP/SQL, and I was able to learn a lot about back-end development in the process.

  • 02. Dark/Light Mode

  • Using JavaScript and JQuery, I implemented a normal and dark mode function in the application. Upon clicking on the toggle, the web application will change themes. In the process of developing this feature, I learned a lot about DOM objects and using JavaScript to modify HTML/CSS elements.
