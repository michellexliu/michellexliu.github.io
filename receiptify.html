  • Receiptify

  • Spotify analytics in a fun, shareable format.

    > Go to website
    > GitHub Repository

  • Inspired by the Instagram account @albumreceipts, this web application will display a user's top tracks from the past month, 6 months, and all time, in "receipt" formats. With over 1,000,000 users in the first 24 hours, Receiptify quickly became viral and was featured in multiple articles globally. This project was built with NodeJS, Express, the JavaScript handlebars.js library, and the Spotify API.
